## Classification Key

| Code | Meaning |
|---|---|
| `VALID_TEACHER_REFERENCE` | Genuinely reproducible from code; used as teacher/reference but not live-capital ready |
| `VALID_EXECUTABLE_BENCHMARK` | Reproducible AND would be a real benchmark if live-tested |
| `TEACHER_REFERENCE` | Reference strategy for distillation/comparison; not fully locked |
| `RESEARCH_ONLY` | Research output; not a locked benchmark |
| `INVALID_FORCED_METRIC` | Metrics were hardcoded/forced; do not use as targets |
| `PARTIAL_RECOVERY` | Progress made but full recovery not achieved |
| `DIAGNOSTIC_ONLY` | Diagnostic/audit output; not a strategy result |
| `ENGINE_PROGRESS` | Engine infrastructure improved; strategy not yet profitable |
| `INFRASTRUCTURE` | Project setup / tooling |
